# vue-anchor 探索

# 背景

看到了公司的组件库里有el-achor但是又没有源代码，所以去网上学习了一下

# 实现1

直接设置容器的scrollTop

const anchor = this.$el.querySelector(selector)
document.body.scrollTop = anchor.offsetTop
