# arco-design快速使用tailwind的颜色

arco design的颜色是它自己定义的，但是我们可以使用tailwind的颜色，这样配合tailwind的颜色工具，可以更方便的使用颜色。

因为是Vue3的项目，所以使用的是unocss的tailwind预设。然后

然后就是css文件，这里我们使用uno-{color}的形式，只要使用了uno-{color}这个类，arco design的primary等颜色就会被替换成tailwind的颜色。

// @unocss-include

export function useColors() {
  const colorList = [
    'red',
    'orange',
    'amber',
    'yellow',
    'lime',
    'green',
    'emerald',
    'teal',
    'cyan',
    'sky',
    'blue',
    'indigo',
    'violet',
    'purple',
    'fuchsia',
    'pink',
    // 'rose',
    // 'slate',
    // 'gray',
    // 'zinc',
    // 'neutral',
    // 'stone',
  ]

  const randomColor = () => {
    return colorList[Math.floor(Math.random() * colorList.length)]
  }
  return {
    colorList,
    randomColor,
  }
}
