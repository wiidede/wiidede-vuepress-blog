### DDRxn寄存器

- 1 = 输出
- 0 = 输入

### PORTxn寄存器

- 1 = 引脚输出高电平 上拉电阻使能
- 0 = 引脚输出低电平 上拉电阻去使能

### PINxn

- 读取引脚电平变化（输入状态）

```c
//pc3输出高电平
DDRC |= 0x08;
PORTC |= 0x08;
//pc4输出低电平
PORTC &= 0xEF;
//pc0134输出高电平
DDRC |= (1 << 0) | (1 << 1) | (1 << 3) | (1 << 4);
//pc2567输出低电平
DDRC &= ~((1 << 2) | (1 << 5) | (1 << 6) | (1 << 7));
```

## USART

UCSRA = 0

UCSRB = 1 << RXCIE | 1 << RXEN | 1 << TXEN

UCSRC = 1 << URSEL | 1 << UCSZ1 | 1<< UCSZ0

UBRRH/UBRRL

u16 ubrr = 383

UBRRL = ubrr

UBRRH = ubrr >> 8
