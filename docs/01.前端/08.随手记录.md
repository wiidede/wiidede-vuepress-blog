# 前端的随手记录

## fit-content

firefox 的 width `fit-content` 目前还需要加前缀

```css
div {
    width: fit-content;
    width: -moz-fit-content;
}
```

**问题来源：**在居中 el-form的时候，可以使用这个，加上`margin: 0 auto`就可以实现居中
