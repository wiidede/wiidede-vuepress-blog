# WebPack 知识总结

## 不打包某个文件夹

比如在复制资源文件下的时候，不打包assests中的js目录

```js
new CopyWebpackPlugin([{
    from: path.resolve(__dirname, `../assets`),
    to: `assets`,
    ignore: ['js/**']
}])
```

`*.txt`: 忽略txt后缀

`.*`: 忽略编辑器文件或特殊文件

`js/**`: 忽略js文件夹

`js/*`: 忽略某js文件夹下内容
